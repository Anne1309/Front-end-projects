# 🔴🟡 Connect 4 Game  

A classic Connect 4 game built using HTML, CSS, and JavaScript.  

## 🚀 Features  
- Two-player mode (PvP)   
- Simple and responsive design  
- Displays message for winner

## 🛠 Tech Stack  
- **Frontend:** HTML, CSS, JavaScript
