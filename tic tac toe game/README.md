# 🎮 Tic-Tac-Toe Game  

A simple, interactive Tic-Tac-Toe game built using HTML, CSS, and JavaScript.  

## 🚀 Features  
- Play against another player (PvP mode)  
- Simple and clean UI   
- Responsive design for mobile and desktop  

## 🛠 Tech Stack  
- **Frontend:** HTML, CSS, JavaScript
